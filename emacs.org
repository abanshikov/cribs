* Сочетания клавиш
| Сочетание клавиш | Команда | Описание                                                     |
|------------------+---------+--------------------------------------------------------------|
| M-x              |         | Перейти к вводу команд                                       |
| C-=              |         | Расширение региона выделения (после установки expand-region) |
| C-;              |         | Поиск по символу на экране (после установки avy)             |
